## Smart Attendance Analyzer
A web-based attendance management system developed using Java Servlets, JSP, MySQL, and Design and Analysis of Algorithms (DAA) concepts.

## Features
* Attendance percentage calculation
* Greedy algorithm for bunk analysis
* Dynamic JSP output
* MySQL database integration
* Input validation and data processing

## Technologies Used
* Java
* Servlets
* JSP
* MySQL
* HTML/CSS/JavaScript
* Apache Tomcat

## Algorithm Used
The project uses a greedy algorithm to calculate the maximum number of classes a student can bunk while maintaining the minimum attendance requirement.
